IEC-62056 es un conjunto de normas internacionales publicadas por la IEC (International Electrotechnical Commission), las cuales definen los protocolos y mecanismos de comunicación para el intercambio de información entre medidores eléctricos y los sistemas centrales.

-- Puede verse como el marco normativo sobre el que esta definido [[DLMS-COSEM]]

La normativa IEC-62056 dentro de su amplitud cubre los siguientes aspectos::
- Protocolo de Comunicación
- Modelo de Datos
- Mecanismos de Seguridad
- Transporte de Datos Sobre Distintos Medios

## Protocolo de Comunicación
DLMS **(Device Language Message Specification)** ([[DLMS-COSEM]]) es el protocolo de comunicación definido dentro dentro del marco normativo IEC-62056. Este es principalmente utilizado para establecer como se se forma y transmiten los mensajes entre el medidor y el sistema central.

## COSEM
COSEM **(Companion Specification for Energy Metering)** es un modelo de datos orientado a objetos definido en IEC62056 para organizar la información obtenida por el medidor en datos estructurados.
