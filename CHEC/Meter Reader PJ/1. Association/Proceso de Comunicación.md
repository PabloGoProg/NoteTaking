### Asociación de Aplicaciones
Las asociaciones de aplicaciones son conexiones lógicas entre un cliente y un servidor DLMS/COSEM. Cuando una petición AA es aceptada por un servidor, se puede asumir lo siguiente:

1. El usuario del cliente es conocido por el servidor.
2. El contexto de asociación propuesto por el cliente es aceptable para el servidor.
3. El mecanismo de autenticación es aceptado por el servidor.
4. Los elementos del contexto xDLMS pueden ser negociados entre el cliente y el servidor.
