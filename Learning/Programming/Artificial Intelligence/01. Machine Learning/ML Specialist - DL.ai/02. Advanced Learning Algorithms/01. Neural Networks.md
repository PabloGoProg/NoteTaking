
## Layer of NN (NNL)

![[Pasted image 20260516111822.png]]

Process of NNL:

1. Each layer has an input vector, $\vec{X}$ if the layer is the first hidden layer, $\vec{a}^{[i]}$ otherwise.
2. Each neuron is a model itself, e.g. the neurons of the image are logistic models itself.
3. The outputs of neurons are the inputs of the next layer ($\vec{a}^{[i]}$).
4. The formula for a single activation value is: $a_{j}^{[l]} = g\left(\vec{w}_{j}^{[l]} \cdot \vec{a}^{[l-1]} + b_{j}^{[l]}\right)$
