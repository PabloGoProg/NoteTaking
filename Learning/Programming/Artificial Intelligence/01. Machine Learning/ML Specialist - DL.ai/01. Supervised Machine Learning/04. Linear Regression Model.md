## Definition

Supervised learning and statistical modeling method that **estimates** the relationship between one or more independent variables `(x1, x2, x3, ..., xn)` and a continuous dependent target `y` by assuming that this relationship can be represented as a linear function plus an error term.

- For just one feature: $f_{w,b}(x) = wx + b$
- For `n` number of features: $f_{w,b}(x) = w_1x_1 + w_2x_2 + ... + w_nx_n + b$
- For `n` number of features (vectorized): $f_{\vec{w},b}(\vec{x}) = \vec{w} \cdot \vec{x} + b$
