Supervised learning algorithms are algorithms that learn from right answers. More technical, algorithms that learn from ``(x, y)`` mappings. -> Once trained, they aim to get never seen samples of ``x`` and predict a valid `y`.

- Learn from mapped pairs `(x, y)`.
- Requieres a labeled dataset.
- Allow the model to _identify patterns_ between the inputs and targets to generalize in unseen data.

## Supervised Learning Major Types

- **Regression**: The _goal_ is to predict a number from an infinite possible output of numbers.
- **Classification**: The goal is to predict a Category or Class of a small set of possible outputs. The type of a category could be anything, range of numbers, things,
