A _Computer Network_ is a telecommunication framework which allow digital devices (nodes) to **interact** (via wired mediums or wireless mediums) to **share resources** (that could be hardware or software).

## Data Communication Components

1. Sender
2. Receiver
3. Message
4. Medium
5. Rules -> Protocols: The main goal of protocols is synchronization > Do something is already agreed upon.

## Networks Effectiveness

Which are the metrics to consider a network effective?

1. **Delivery**: Data is delivered to the correct destination.
2. **Integrity** | Accuracy: Data should not be modified between the **transmission** | **Accurately** delivered without errors.
3. **Timeliness**: Data must be delivered in a timely manner.
4. **Jitter**: Uneven delay.
