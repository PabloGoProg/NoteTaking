**What is Spec-Driven Development**

_What is the specification_

The specification is a precise, descriptive and structured description of how a system should behave. The _goal_ of the specification is to reduce ambiguity and left less space to model interpretations.




**What's the difference between SDD and _Vibe Coding_?**

Vibe coding is ==a software development approach== where humans build applications by guiding AI assistants through conversational prompts rather than writing code line-by-line.

- The developer acts as a ==product manager or conductor==, describing functionality in plain natural language ==while AI handles the code generation==, debugging, and, in some cases, deployment.
- The process emphasizes what the app should _do_ and _feel like_, rather than the specific or technical syntax.
