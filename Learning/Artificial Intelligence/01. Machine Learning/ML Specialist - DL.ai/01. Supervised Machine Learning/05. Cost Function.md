A cost function is a mathematical function that _quantifies the discrepancy between a model’s predicted outputs and the true target values_ over a training dataset.

$$ J(\theta) = \frac{1}{m}\sum_{i=1}^{m} L(y_i, \hat{y}_i) $$
Where:

- $J(\theta)$: The cost function value for the model with parameters $\theta$.
- $m$: Total number of data points in the dataset.
- $y_i$: Original target values.
- $\hat{y}_i$: Predicted values.
- $L(y_i, \hat{y}_i)$: Loss function that computes the error for a single data point.
