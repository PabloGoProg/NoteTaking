Arthur Samuel says that Machine Learning "Field of study that gives computers the ability to learn without being explicitly programmed."

## Major Types of Machine Learning

1. [[02. Supervised Learning]]: Paradigm in which an algorithm learns a **mapping** from input variables to _target output variables_ using a **labeled dataset**. Each training example contains both the input data and the correct output `(x, y)`, allowing the model to learn patterns that generalize to unseen data. Common tasks include classification and regression.
2. [[03. Unsupervised Learning]]: Paradigm in which an algorithm analyzes data without labeled target outputs. The model seeks to identify hidden structure, patterns, groupings, distributions, or representations within the input data itself. Common tasks include clustering, dimensionality reduction, anomaly detection, and association discovery.
