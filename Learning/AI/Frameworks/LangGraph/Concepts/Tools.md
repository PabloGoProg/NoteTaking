Tools are _specialized functions or utilities_ that nodes ([[Node]]) can utilize to perform specific tasks such as fetching data from an API.

- They enhance the capabilities of nodes by providing additional functionalities to them.
- Nodes are parts of the graph structure while tools are functionalities within nodes.

## Tool Nodes
Tool Nodes are a special kind of node whose main job is to run a tool. -> It connects the tool's output back into the state, so other nodes can use that information.
