Un prompt es una solicitud/instrucción que se le da a una IA.

 - La calidad de la respuesta generada incrementa en función de la calidad de pregunta.

---
**Prompt Shadowing** -> Es el proceso mediante el que ajustamos la salida hasta obtener lo esperado (incluyendo formato, tono, enfoque y referencias). -> Es valioso solicitar una reformulación del prompt original para obtener el que te lleve al resultado esperado.

**Prompt Roles** -> Asignar un rol a un modelo desplaza el espacio vectorial hacia el conocimiento relevante, activando patrones de lenguaje específicos aprendidos durante el entrenamiento.

**Prompt Structuring** -> Es una guía para detectar falencias, pequeños prompts pueden obtener resultados buenos, pero no consistentes.
