Requirements
1. GTFA - (Ground Truth Final Answer): The prompt should have **only one** objectively verifiable correct answer.
2. Anchored In Time: The answer should not change over time unless the source data is fundamentally wrong.
3. Requires a Tool Call: The question should require **using a tool** (e.g., web search, database query, API call) to answer.
4. Is not combining too many Asks: The prompt should make no more than 2–3 asks in total.
5. Does Not Prescribe a Tool: The prompt should not name or strongly imply a specific tool, service, or source.
6. Natural, Not Contrived: The question should read naturally and not be artificially complex. Are complex enough to require at least 2 reasoning steps
